Технології

Node.js є платформою, побудованою на Chrome's JavaScript runtime для простої побудови швидких, масштабованих мережних додатків.

Експрес є мінімальною та гнучкою структурою node.js веб-додатків, надаючи широкий набір функцій для побудови одно- та багатосторінкових, як і гібридних веб-додатків.

MongoDB є провідною NoSQL базою даних, що розширює можливостей бізнесу, аби бути більш гнучкою і масштабованою.

AngularJS дозволяє розширити словниковий запас HTML для вашого застосування. Отримане середовище надзвичайно виразне, що читається, і швидко розвивається.
