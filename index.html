Технології

Node.js є платформою, побудованою на Chrome's JavaScript runtime для простої побудови швидких, масштабованих мережних додатків.

Експрес є мінімальною та гнучкою структурою node.js веб-додатків, надаючи широкий набір функцій для побудови одно- та багатосторінкових, як і гібридних веб-додатків.

MongoDB є провідною NoSQL базою даних, що розширює можливостей бізнесу, аби бути більш гнучкою і масштабованою.

AngularJS дозволяє розширити словниковий запас HTML для вашого застосування. Отримане середовище надзвичайно виразне, що читається, і швидко розвивається.

Screen


Screen is nice one page theme implementing one page scroll. You can use it in many ways - to build portfolio, tell a story, present your pictures. This Free Bootstrap template is built with Bootstrap 3.3.2, so fully responsive and should be working well on majority of the devices out there :)

Universal


Universal is clean and stylish universal website template built with Bootstrap 3.3. It stands out with its clean design and elegant typography. It is one of the most complex free Bootstrap templates and with almost 30k downloads (as of Feb 2017), it is also my most popular Bootstrap freebie.

Black and White


Black and White is minimalistic theme based on great images and clean typography. It certainly could be used to build cool portfolio or classy landing page. This one page Bootstrap theme includes: - full screen intro image, - sticky navbar, - services section, - portfolio or gallery section with lightbox, - contact form , - custom style Google Map (grayscale theme).

IT Worker


IT Worker could be your new perfect portfolio template. It contains all the necessary elements for successful portfolio, is based on one of the latest Bootstrap versions and is fully responsive.
